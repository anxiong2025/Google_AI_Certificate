# 实践项目指南

基于 Google AI 专业证书课程，精选 **5 个实战项目**，分三级难度，每个项目都有完整 Prompt 模板，打开就能跟着做。

## 项目地图

```
🟢 入门篇 — 熟悉工具，建立信心
├── 项目 A：Prompt 工程实验室        ← 掌握与 AI 对话的核心技能
│
🟡 进阶篇 — 解决真实问题
├── 项目 B：个人知识库 + 研究助手    ← 构建你的第二大脑
├── 项目 C：数据分析实战             ← 用自然语言做数据分析
│
🔴 综合篇 — 端到端交付
├── 项目 D：AI 内容营销全案          ← 从品牌定位到素材交付
└── 项目 E：Vibe Code 一个实用工具   ← 零代码基础也能做应用
```

## 快速导航

| 难度 | 项目 | 耗时 | 工具 | 适合谁 |
|------|------|------|------|--------|
| 🟢 | [A. Prompt 工程实验室](A_Prompt工程实验室.md) | 1-2h | Gemini | 所有人，必做 |
| 🟡 | [B. 个人知识库](B_个人知识库.md) | 2-3h | NotebookLM / Deep Research / Gems | 需要做研究的人 |
| 🟡 | [C. 数据分析实战](C_数据分析实战.md) | 2-3h | Gemini in Sheets | 需要处理数据的人 |
| 🔴 | [D. AI 内容营销全案](D_AI内容营销全案.md) | 3-4h | Gemini / Canvas / Slides | 做内容、运营的人 |
| 🔴 | [E. Vibe Code 实用工具](E_VibeCoding实用工具.md) | 2-3h | Google AI Studio | 想做应用的人 |

## 开始之前

**你需要准备**：
- Google 账号（用于 Gemini、NotebookLM、Sheets、Slides）
- Gemini 访问权限（免费版即可，Pro 效果更好）
- Google AI Studio 账号（项目 E 需要）

**建议顺序**：A → B → C → D → E（按依赖关系排列）
