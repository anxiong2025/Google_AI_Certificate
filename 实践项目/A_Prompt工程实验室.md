# 🟢 项目 A：Prompt 工程实验室

> **难度**: 入门 ｜ **耗时**: 1-2 小时 ｜ **工具**: Gemini  
> **对应课程**: 01 AI Fundamentals — Module 04, 05

## 为什么做这个项目

Prompt 是使用所有 AI 工具的基础。写好 Prompt 和写不好，结果差距巨大。这个项目用 6 个递进实验，让你从 "随便问问" 升级到 "精准控制 AI 输出"。

---

## 实验 1：基础 Prompt vs 优化 Prompt

**目标**: 感受 Prompt 质量对结果的影响

先用模糊 Prompt，再用结构化 Prompt，对比输出差异。

### 操作

**第 1 步 — 模糊提问**（直接复制到 Gemini）

```
帮我写个邮件
```

**第 2 步 — 结构化提问**

```
你是一位专业的商务沟通顾问。

请帮我写一封邮件，要求如下：
- 场景：向客户介绍我们新推出的 AI 数据分析服务
- 语气：专业但友好，不要过于销售化
- 结构：开头问候 → 服务亮点（3 点）→ 邀请试用 → 结尾
- 长度：200 字以内
- 语言：中文
```

### 观察要点
- 对比两次输出的质量差异
- 注意结构化 Prompt 的 5 个要素：**角色、场景、语气、结构、约束**

## 实验 2：角色扮演的力量

**目标**: 学会用 "角色设定" 控制 AI 的专业深度

用同一个问题，分别给 AI 设定不同角色，观察输出差异。

### 操作

**问题**: "如何提高团队效率？"

**角色 A — 管理咨询师**
```
你是麦肯锡的资深管理咨询师，有 15 年帮助企业优化团队效率的经验。

客户问你：如何提高一个 10 人技术团队的工作效率？

请给出你的专业建议，包含：
1. 诊断框架（先分析什么）
2. Top 3 改进措施
3. 预期效果和时间线
```

**角色 B — 心理学家**
```
你是一位组织心理学教授，专注研究团队动力学和员工激励。

一个团队管理者问你：如何提高一个 10 人技术团队的工作效率？

请从心理学角度给出建议，包含：
1. 影响效率的心理因素
2. 具体的行为干预方法
3. 需要避免的常见误区
```

### 观察要点
- 同一问题，不同角色给出完全不同的视角
- 角色越具体（行业 + 年限 + 专长），输出越专业

## 实验 3：Few-shot 示例法

**目标**: 学会用示例引导 AI 模仿你的风格

### 操作

```
我需要你帮我写产品更新日志，请模仿以下风格：

【示例 1】
🚀 v2.1 — 更智能的搜索
搜索终于不再"装傻"了。现在它能理解你真正想找什么，即使你打了错别字。
- 语义搜索上线，告别关键词匹配时代
- 错别字自动纠正，手滑也不怕
- 搜索速度提升 40%，快到你反应不过来

【示例 2】
🎨 v2.0 — 全新的深色模式
终于，你的眼睛可以休息了。深色模式不只是把白变成黑——我们重新设计了每个像素。
- 3 套深色主题可选（经典黑 / 深蓝 / 暗灰）
- 自动跟随系统设置
- 代码高亮针对深色背景重新调色

---

现在请用相同风格，写 v2.2 版本的更新日志：
- 新功能：AI 写作助手
- 改进：支持导出 PDF
- 修复：iOS 端偶尔闪退
```

### 观察要点
- AI 会精确模仿你的语气、emoji 风格、段落结构
- 给 2-3 个示例效果最佳，比纯文字描述风格高效得多

## 实验 4：提示链（Prompt Chaining）

**目标**: 学会把复杂任务拆成多步，逐步深入

场景：为一个新产品做竞品分析。

### 操作

**第 1 步 — 定义分析框架**
```
我要对"AI 笔记应用"这个赛道做竞品分析。

请帮我设计一个竞品分析框架，包含：
- 需要分析哪些维度（至少 6 个）
- 每个维度的评分标准
- 输出格式建议
```

**第 2 步 — 填充竞品数据**（基于第 1 步的框架）
```
基于上面的框架，请分析以下 5 个竞品：
Notion AI、Obsidian + AI插件、NotebookLM、Mem.ai、Reflect

请用表格形式呈现，每个维度用 1-5 分评分。
```

**第 3 步 — 生成洞察**（基于第 2 步的数据）
```
基于上面的竞品分析表格，请：
1. 总结各产品的核心差异化优势
2. 找出市场空白（哪些需求没被很好满足）
3. 如果我要做一个新产品，给出 3 个差异化定位建议
```

### 观察要点
- 每一步都建立在前一步的结果上
- 拆解后的输出质量远好于一次性提问
- 这就是 **提示链**：step-by-step 引导 AI 完成复杂任务

## 实验 5：输出格式控制

**目标**: 精确控制 AI 的输出格式

### 操作

用同一个话题（"远程办公的优缺点"），要求不同输出格式：

**格式 A — Markdown 表格**
```
请用 Markdown 表格列出远程办公的 5 个优点和 5 个缺点。
表格列：序号 | 类别（优点/缺点）| 内容 | 影响程度（高/中/低）
```

**格式 B — SWOT 分析**
```
请对"企业全面推行远程办公"做 SWOT 分析。
用 2x2 矩阵格式输出，每个象限列 3-4 点。
```

**格式 C — JSON 数据**
```
请将远程办公的分析结果输出为 JSON 格式，结构如下：
{
  "topic": "远程办公",
  "pros": [{"point": "...", "impact": "high/medium/low", "detail": "..."}],
  "cons": [{"point": "...", "impact": "high/medium/low", "detail": "..."}],
  "recommendation": "..."
}
```

### 观察要点
- AI 可以输出几乎任何格式：表格、JSON、XML、CSV、Mermaid 图
- 明确指定格式 + 给出结构示例 = 精确控制输出

## 实验 6：Prompt 调试——当 AI 输出不理想时

**目标**: 学会诊断和修复 Prompt

### 常见问题及修复策略

| 问题 | 原因 | 修复方法 |
|------|------|----------|
| 输出太笼统 | 缺少具体约束 | 加上"请举具体例子"、"请用数据支撑" |
| 输出太长 | 没限制长度 | 加上"请在 X 字以内"、"请只列 Top 3" |
| 风格不对 | 没定义语气 | 加上角色设定或给出示例 |
| 偏离主题 | 问题太发散 | 缩小范围，一次只问一个点 |
| 内容有误 | AI 幻觉 | 要求"如不确定请标注"、提供参考资料 |

### 操作 — 调试练习

**原始 Prompt**（故意写得很差）：
```
帮我分析一下市场
```

**逐步优化**：

```
# 第 1 次优化：加具体领域
帮我分析中国新能源汽车市场

# 第 2 次优化：加时间范围和角度
帮我分析 2024-2025 年中国新能源汽车市场的竞争格局

# 第 3 次优化：加角色和输出要求
你是一位汽车行业分析师。请分析 2024-2025 年中国新能源汽车市场的竞争格局。

要求：
- 按价格带（10万以下/10-20万/20-30万/30万以上）分析
- 每个价格带列出 Top 3 品牌及市场份额
- 指出各价格带的竞争趋势
- 用表格呈现，最后给出 3 个关键洞察
- 内容基于公开可查的数据，不确定的地方请标注
```

### 观察要点
- 每次优化增加一个维度，输出质量阶梯式上升
- 好的 Prompt = **角色 + 任务 + 背景 + 约束 + 格式**
