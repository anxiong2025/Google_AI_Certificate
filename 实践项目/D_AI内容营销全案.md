# 🔴 项目 D：AI 内容营销全案

> **难度**: 综合 ｜ **耗时**: 3-4 小时 ｜ **工具**: Gemini / Gemini Canvas / Gemini in Slides  
> **对应课程**: 04 Writing & Communicating, 05 Content Creation

## 为什么做这个项目

一套完整的营销案从策划到交付，通常需要文案、设计师、PPT 美化多人协作。这个项目让你一个人 + AI 完成全流程，产出一套可以直接用的品牌营销素材。

---

## 选择你的品牌

选择以下场景之一（推荐 A 或 B，离自己近的更有动力做）：

| 场景 | 示例 | 难度 |
|------|------|------|
| A. 个人品牌 | 自己的自媒体/作品集/个人网站 | ⭐⭐ |
| B. 虚拟创业项目 | 一个你想做的产品/服务 | ⭐⭐ |
| C. 已有产品改造 | 为公司某产品重新做一版营销 | ⭐⭐⭐ |

---

## Part 1：品牌定位（30 min）

### 第 1 步 — 用 Gemini 头脑风暴

```
我想为 [你的产品/品牌/个人品牌] 做一套完整的营销方案。

背景信息：
- 产品/服务：[简述是什么]
- 目标用户：[你认为的目标人群]
- 核心价值：[你觉得最大的卖点]

请帮我做品牌定位分析：
1. 目标用户画像（年龄、职业、痛点、需求）
2. 3 个差异化定位方向（附理由）
3. 品牌调性建议（正式 vs 活泼、专业 vs 亲民等）
4. 一句话品牌口号（提供 5 个选项）
5. 竞品参考（类似的品牌是怎么定位的）
```

### 第 2 步 — 选定方向并深化

```
我选择定位方向 [X]，品牌口号用"[你选的]"。

请进一步完善：
1. 品牌故事（200 字，有情感共鸣）
2. 核心信息架构：
   - 一句话介绍（电梯演讲版）
   - 三句话介绍（社交媒体版）
   - 一段话介绍（官网版，100 字）
3. 品牌关键词 5 个（用于贯穿所有营销内容）
```

---

## Part 2：多版本文案（40 min）

### 第 1 步 — 用 Canvas 写作

打开 Gemini Canvas（画布模式），创建不同受众的文案：

**客户版文案**
```
请为 [产品] 写一篇面向目标客户的介绍文案。

品牌信息：[粘贴 Part 1 的品牌故事和关键词]
目标读者：[目标用户画像]
用途：官网首页 / 产品详情页
语气：[你选的品牌调性]
长度：300 字
结构：痛点引入 → 解决方案 → 核心优势（3 点）→ 行动号召

要求：
- 第一句话就要抓住注意力
- 用具体数字或场景代替空泛描述
- 以用户视角而非产品视角写
```

**社交媒体版**
```
请为 [产品] 写 5 条社交媒体推文：

平台：小红书 / LinkedIn / Twitter（各写风格不同的版本）
品牌调性：[同上]

要求：
1. 小红书版：口语化、有emoji、300字以内、有话题标签
2. LinkedIn版：专业、有洞察、分享行业思考、200字以内
3. Twitter/X版：简短有力、120字以内、带互动提问

每个平台写 2 条，主题分别是：
- 产品功能介绍
- 用户痛点共鸣
```

### 第 2 步 — 压力测试文案

```
请以营销专家的身份审查以下文案，并给出严格的反馈：

[粘贴你的文案]

请评估：
1. 开头是否足够吸引人？（1-10 分）
2. 价值主张是否清晰？用户看完知道"这跟我有什么关系"吗？
3. 行动号召是否明确有力？
4. 有没有废话可以删掉？
5. 竞品如果看到这个文案会怎么评价？

然后请直接给出改进后的版本。
```

## Part 3：视觉素材创建（40 min）

### 第 1 步 — 用 Gemini 生成图片

```
请为 [产品/品牌] 生成以下营销图片：

品牌调性：[例如：科技感、简约现代、温暖亲和]
主色调：[例如：深蓝 + 白色]

需要生成：
1. 品牌主视觉（适合官网 banner，展示产品核心场景）
2. 社交媒体配图（正方形，适合 Instagram/小红书）
3. 产品功能展示图（信息图风格，展示 3 个核心功能）
```

### 第 2 步 — AI 设计评审

```
请以平面设计总监的身份评审以下设计方案：

[描述或上传你的视觉素材]

评审维度：
1. 品牌一致性（是否传达了正确的调性？）
2. 视觉层次（信息重点是否突出？）
3. 色彩运用（是否协调？是否适合目标人群？）
4. 排版（文字可读性、留白是否合理？）
5. 整体印象（1-10 分）

请给出具体改进建议，特别是哪些地方可以让非设计师也能调整。
```

## Part 4：演示幻灯片（40 min）

### 第 1 步 — 在 Google Slides 中用 Gemini 生成

打开 Google Slides，用 Gemini 助手：

```
请帮我创建一份品牌介绍 PPT，共 8 页：

第 1 页：封面（品牌名 + 口号 + 主视觉）
第 2 页：问题/痛点（用户面临什么问题？）
第 3 页：解决方案（我们怎么解决？一句话 + 配图）
第 4 页：核心功能/优势（3 个卖点，每个配图标）
第 5 页：使用场景（2-3 个具体使用场景）
第 6 页：用户评价/数据证明（社会证明）
第 7 页：定价/方案对比（如果适用）
第 8 页：行动号召 + 联系方式

风格：[简约/科技/温暖]，配色：[你的品牌色]
每页文字不超过 30 个字，多用视觉表达
```

### 第 2 步 — 优化排版和视觉

```
请审查这份 PPT 并优化：

1. 每页的文字是否太多？哪些可以删减？
2. 视觉层次建议（字号、加粗、颜色对比）
3. 哪几页最需要加配图或图标？
4. 整体故事线是否流畅？页面顺序需要调整吗？
5. 如果只看每页标题，能否理解完整故事？
```

---

## Part 5：整合交付（20 min）

### 最终素材清单

```
请帮我整理一份交付清单，确保品牌营销套件的完整性：

已完成：
☐ 品牌定位文档（品牌故事 + 口号 + 关键词）
☐ 客户版文案（官网/详情页）
☐ 社交媒体文案（小红书/LinkedIn/Twitter 各 2 条）
☐ 营销图片（3 张以上）
☐ 品牌介绍 PPT（8 页）
☐ 设计评审报告

请检查以上内容的品牌一致性：
- 口号和关键词是否贯穿所有素材？
- 语气风格是否一致？
- 视觉色调是否统一？
- 如果有不一致的地方，请指出并给出修改建议。
```

- [ ] 品牌定位文档（故事 + 口号 + 画像 + 关键词）
- [ ] 3 版文案（官网版 + 社交媒体版 + 压力测试修改版）

- [ ] 8 页品牌介绍 PPT
- [ ] 设计评审报告
- [ ] 品牌一致性检查结果
