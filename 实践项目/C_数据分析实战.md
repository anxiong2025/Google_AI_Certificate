# 🟡 项目 C：数据分析实战

> **难度**: 进阶 ｜ **耗时**: 2-3 小时 ｜ **工具**: Gemini / Gemini in Google Sheets  
> **对应课程**: 06 Data Analysis — 全部

## 为什么做这个项目

不需要学 SQL 或 Python，用自然语言就能做数据清洗、分析和可视化。这个项目用一个真实数据集，走一遍完整的数据分析流程。

---

## 准备数据

选择以下任一数据集（推荐选项 A，最贴近个人生活）：

| 选项 | 数据来源 | 难度 | 说明 |
|------|----------|------|------|
| A | 个人消费记录 | ⭐ | 导出支付宝/微信/银行账单 CSV |
| B | 公开数据集 | ⭐⭐ | [Kaggle Datasets](https://www.kaggle.com/datasets) 选一个感兴趣的 |
| C | 模拟销售数据 | ⭐ | 用下方 Prompt 让 Gemini 生成 |

**如果没有现成数据，用 Gemini 生成模拟数据：**

```
请帮我生成一份模拟的电商销售数据，要求：

格式：CSV（可以直接粘贴到 Google Sheets）
行数：200 行
列：订单日期, 产品类别, 产品名称, 单价, 数量, 总金额, 客户地区, 支付方式, 是否退货

要求：
- 日期范围：2025-01-01 到 2025-12-31
- 产品类别：电子产品、服装、食品、家居、图书
- 客户地区：北京、上海、广州、深圳、成都、杭州
- 数据要有一定规律（如双11前后销量高、电子产品客单价高）
- 退货率约 8%
- 包含一些"脏数据"（缺失值、格式不一致）用于清洗练习
```

## Part 1：定义 KPI（20 min）

在 Gemini 中进行，确定这份数据应该关注什么。

### 操作

```
我有一份电商销售数据（或：个人消费数据），包含以下字段：
[列出你的数据列名]

数据时间范围：[填写]
数据量：约 [X] 行

请帮我：
1. 定义 5 个最重要的 KPI（关键绩效指标），并说明每个 KPI 的业务意义
2. 对每个 KPI，给出计算公式
3. 建议用什么图表类型展示每个 KPI
4. 按重要性排序

请用表格格式输出。
```

将 Gemini 的回答保存下来，这是你后续分析的框架。

---

## Part 2：数据清洗（30 min）

### 第 1 步 — 导入数据到 Google Sheets

1. 打开 [Google Sheets](https://sheets.google.com/)
2. 新建表格
3. 粘贴或导入 CSV 数据

### 第 2 步 — 用 Gemini in Sheets 检查数据质量

在 Sheets 侧边栏打开 Gemini，使用以下 Prompt：

```
请检查这份数据的质量问题：
1. 有哪些列存在缺失值？各缺失多少行？
2. 数据格式是否一致（比如日期格式、金额格式）？
3. 是否有明显的异常值或不合理数据？
4. 请列出所有发现的问题及修复建议。
```

### 第 3 步 — 执行数据清洗

```
请帮我清洗这份数据：
1. 统一日期格式为 YYYY-MM-DD
2. 金额列确保都是数字格式，去掉货币符号
3. 缺失的地区用"未知"填充
4. 删除总金额为负数或零的行（退货行保留但标记）
5. 产品类别统一用简体中文
```

### 第 4 步 — 验证清洗结果

```
数据清洗完成后，请给我一份数据概览：
- 总行数
- 每列的数据类型和非空数量
- 数值列的基本统计（最小值、最大值、平均值、中位数）
- 分类列的唯一值和各类别数量
```

---

## Part 3：数据分析（40 min）

### 第 1 步 — 基础分析

在 Gemini in Sheets 中逐个提问：

```
请按月汇总销售总金额，并告诉我：
1. 哪个月销售额最高？最低？
2. 环比增长率如何变化？
3. 是否有明显的季节性规律？
```

```
请按产品类别分析：
1. 每个类别的总销售额和订单量
2. 每个类别的平均客单价
3. 哪个类别的退货率最高？
4. 请用柱状图展示各类别销售额对比
```

```
请按客户地区分析：
1. 各地区的销售额排名
2. 各地区的客户偏好（最常购买的类别）
3. 各地区的平均客单价有差异吗？
```

### 第 2 步 — 进阶分析

```
请做一个交叉分析：地区 × 产品类别 的销售额矩阵
- 用热力图或条件格式标注高/低值
- 找出每个地区的"明星产品"和"弱势类别"
```

```
请分析支付方式：
1. 不同支付方式的使用比例
2. 不同支付方式的平均客单价是否有差异
3. 支付方式和退货率是否有关联
```

---

## Part 4：构建仪表盘（40 min）

### 第 1 步 — 创建图表

在 Google Sheets 中创建以下图表（可用 Gemini 辅助）：

```
请帮我创建以下图表：

1. 月度销售趋势 — 折线图（X 轴：月份，Y 轴：销售额）
2. 类别销售占比 — 饼图
3. 地区销售对比 — 柱状图
4. 日均订单量分布 — 面积图
5. KPI 卡片 — 在顶部显示关键数字（总销售额、总订单数、平均客单价、退货率）
```

### 第 2 步 — 搭建仪表盘页面

1. 在 Sheets 中新建一个标签页，命名为 "Dashboard"
2. 将图表移到此页面，合理布局
3. 顶部放 KPI 卡片，下方放图表

### 第 3 步 — 生成分析报告

回到 Gemini，整合发现：

```
基于以上所有分析，请生成一份简洁的数据分析报告：

结构：
1. 数据概览（一句话总结数据规模和时间范围）
2. 关键指标摘要（KPI 表格）
3. 核心发现（Top 5 洞察，每个用一句话 + 数据支撑）
4. 值得注意的问题（异常现象或风险点）
5. 行动建议（基于数据的 3 条可执行建议）

风格要求：简洁、用数据说话、避免废话
```
